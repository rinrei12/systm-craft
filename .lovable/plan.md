

## Personal Automation Portfolio Website

A clean, professional SaaS-style portfolio website with the following sections:

### 1. Hero Section
Full-width hero with headline, subheadline, and CTA button ("View Automation Systems") that scrolls to the portfolio section. Subtle gradient background using the brand colors.

### 2. Services Section
Grid of 8 service cards with icons showing: Lead capture automation, Missed call recovery, Automated lead follow-up, CRM pipeline automation, Appointment booking, AI chat assistants, Client onboarding, eCommerce recovery.

### 3. Automation Systems Portfolio
7 detailed project cards, each showing industry tag, business problem, automation solution summary, and business impact bullets:
- Missed Call Recovery System
- Dead Database Revival System
- No-Show Recovery Automation
- Abandoned Cart Recovery System
- AI Client Onboarding Automation
- AI Facebook Chat Agent
- Automated Lead Follow-Up System

### 4. Workflow Architecture Example
Visual step-by-step flow diagram showing the Missed Call Recovery workflow (Missed Call → Twilio Webhook → n8n Workflow → Lead Capture → SMS Response → CRM Update → Booking Link).

### 5. Tech Stack Section
Icon/label grid for: n8n, GoHighLevel, Zapier, Make, Twilio, WhatsApp, Telegram, Airtable, Google Sheets, SendGrid/Mailgun.

### 6. Contact Section
CTA text with a simple contact form (name, email, message) and email link fallback.

### 7. Navigation
Sticky top navbar with smooth scroll links to each section.

### Design
- **Fonts**: Inter for headings, Source Sans Pro for body (via Google Fonts)
- **Colors**: Primary #2563EB, Secondary #0F172A, Accent #10B981, Background #F8FAFC
- **Style**: Minimal SaaS consulting aesthetic, clean spacing, subtle hover animations, fully responsive
